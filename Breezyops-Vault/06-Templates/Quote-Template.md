# Quote / Proposal Template (B2B)

## Layout
- **Cover block:** brand header · "Proposal for {{customer_name}}" · quote no + date + validity ({{valid_days}} days).
- **Context:** one-paragraph summary of the survey findings (merged from the survey job's summary).
- **Scope table:** per-site sections → line items (service, qty, rate, amount).
- **Commercials:** subtotal, {{#if gst_applicable}}GST{{/if}}, total; payment terms; exclusions list.
- **SLA:** response times, visit windows.
- **Why Breezyair:** 15+ yrs experience, certified, before/after evidence gallery (2 photos).
- **Acceptance:** signature block + "Accept" link.

## Rules
- Prices snapshot at creation (catalog changes don't mutate sent quotes).
- New negotiation round → new version; prior versions locked.
