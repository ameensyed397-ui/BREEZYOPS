# UF6 — Admin Day (Asad)

## Flow
1. Morning dashboard: overnight leads, today's schedule, overdue invoices, SLA flags.
2. Triage inbox (5 min) → book/assign.
3. Field work via [[UF5-Technician-Day]] (Asad is also the technician today).
4. Evening: mark payments, glance at pipeline aging, approve any quotes.
5. Weekly: reports review ([[F12-Reports-and-Analytics]]), broadcast planning, catalog/price tweaks.

## Design principle
Everything in ≤3 taps from the dashboard — Asad runs this between jobs, on a phone.
