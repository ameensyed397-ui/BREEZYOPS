# SOP — Photo Capture

## Categories (must be tagged correctly — they auto-file per client)
- **before** — condition on arrival (unit face, filter, coils, any damage).
- **after** — same angles post-service for a true comparison.
- **issue** — the specific fault (leak point, burnt component, gas gauge reading).

## Standards
- Landscape, well-lit, in focus; include the unit's surroundings once for context.
- Same angle for before/after pairs (this is what makes the IG carousel work).
- Never include people/faces or neighbouring homes' interiors without consent.
- Minimum set: 2 before + 2 after per job; issues as needed.
